Модуль маскировки банковских карт и счетов

Проект представляет собой Python-модуль для маскировки номеров банковских карт
и счетов согласно требованиям безопасности. 

Реализованы функции для обработки как чистых номеров, так и строк формата
"Тип Номер".

История разработки
Базовая реализация - функции маскировки карт и счетов
Интеграция - добавление строк с данными
Финальная доработка - исправление стиля и документации

Особенности реализации
Строгая типизация (аннотации типов для всех функций)
Полное покрытие тестами
